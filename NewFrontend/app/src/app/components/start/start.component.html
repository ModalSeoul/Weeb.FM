Scrobbling

They're a few ways to scrobble at the moment:

  • Pandora
  • Google Play
  • Bandcamp
  • MusicBee
  • Foobar2000
  • MPD (Music Player Daemon)

Don't see your player here?

No problem! We also have RipFM which scrobbles directly from Last.fm to your WILT profile! This is a temporary solution to allow for site development to continue while users can scrobble from any player they desire.

What I Listen To - Open Source

Because the owner likes to pretend he's a nice guy, the source code for WILT and all of the tools it uses is available on his Github page.

The following programs are written in Python and compiled using Pyinstaller

Some users have told me that their anti-virus software has flagged these as potential malware. Obviously that's not the case, but if you're still worried, you're free to run the code available on my Github without downloading anything!
